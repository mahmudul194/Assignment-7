# 👥 KeenKeeper — Keep Your Friendships Alive

**KeenKeeper** is a personal friendship management dashboard designed to help you nurture your most important relationships. It tracks your interactions, sets connection goals, and provides analytics to ensure nobody falls through the cracks.

## 🚀 Key Features

- **Personalized Friend Dashboard**: View all your connections at a glance with status indicators (On-track, Almost Due, Overdue) based on your last contact.
- **Smart Timeline**: Automatically log calls, texts, and video chats with a persistent interaction history and filtering system.
- **Friendship Analytics**: Visualize your social habits with detailed charts showing your preferred interaction methods over time.

## 🛠️ Technologies Used

- **React.js**: Core UI framework for a responsive and dynamic experience.
- **Tailwind CSS 4.0**: State-of-the-art styling with a custom forest green design system.
- **React Router 7**: Seamless navigation between Home, Details, Timeline, and Stats pages.
- **Recharts**: Beautiful, interactive data visualization for friendship analytics.
- **Lucide React**: Clean and consistent iconography.
- **React Hot Toast**: Instant feedback with elegant toast notifications.
